data_num.xlsx是各课程量化后的数据.其中:
第一列是课程会在一周中的周几上课,一共分为7天,即周一到周日,0代表某天没有课程,1代表有课程;
第二列是每周上课节次;
第三列是上课的那一天的哪一节的课程,0代表没有课程,1代表有课程;
第四列是1个学期中哪一天会有课程,列表有18位数,一共有17个教学周,0代表某周没有课程,1代表某周会有课程,第18位固定为0;
第五列是哪个年级开设的课程,一共有大一到大四四个年级,0代表某年级没有此课程,1代表有此课程;
第六列是某节课的选课人数;
第七列是通过某种方式生成的课程的哈希值;
第八列是课程的上课地点,列表一共有7位,其中前2位代表上课地点,3到6位代表某一楼层,最后一列代表特殊地点.

data_num_b.xlsx是个课程的量化前的某些数据,可做补充,data_num.xlsx里的数据按行对应.

data_hash.xlsx是某个课程的哈希值,和data_num.xlsx里的数据按行对应.
